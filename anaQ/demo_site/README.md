# anaQ 系統展示頁面

## 📖 簡介

這是 anaQ 質性編碼整合系統的展示頁面，以互動式網頁的形式呈現系統的功能、架構、編碼框架、分析流程和實際測試結果。

## 🚀 使用方法

### 方法一：直接在瀏覽器中開啟

1. 找到 `demo_site/index.html` 檔案
2. 使用瀏覽器（Chrome、Firefox、Safari、Edge 等）開啟
3. 點擊上方導航標籤瀏覽不同內容區塊

### 方法二：使用本地伺服器（推薦）

使用本地伺服器可以確保所有功能正常運作，特別是互動式圖表。

**Python 3 方式：**
```bash
cd demo_site
python3 -m http.server 8000
```

然後在瀏覽器中訪問：`http://localhost:8000`

**Python 2 方式：**
```bash
cd demo_site
python -m SimpleHTTPServer 8000
```

**Node.js 方式（需先安裝 http-server）：**
```bash
npm install -g http-server
cd demo_site
http-server -p 8000
```

## 🎯 互動功能

### 互動式時間軸（Interactive Timeline）
系統提供多個版本的互動式時間軸視圖，展示課堂對話的時間序列：
- **版本1 & 版本2**：不同測試數據的時間軸分析
- 每個語段的編碼標籤和詳細信息
- 時間軸上的視覺化標記
- 互動式懸停查看詳細資訊
- 可縮放和可拖曳的時間軸視圖

### 時序儀表板（Temporal Dashboard）
提供多維度的時序分析視圖（多個版本）：
- 認知發展軌跡圖
- 情感變化趨勢圖
- 編碼分佈熱力圖
- 關鍵時刻標記
- 互動式圖表放大縮小

### 綜合分析摘要（Analysis Summary）
整合所有分析結果的綜合報告：
- 整體統計數據
- 關鍵發現摘要
- 教學建議
- 可視化圖表集合
